## Prerequisites

- **macOS or Linux** (x64 or ARM)
- **[Docker](https://www.docker.com/products/docker-desktop)** — for LocalNet and Vault
- **[Claude Code](https://docs.anthropic.com/en/docs/claude-code)** or **[OpenCode](https://opencode.ai)** — your AI coding tool

[AlgoKit CLI](https://github.com/algorandfoundation/algokit-cli) is also required, but VibeKit will install it for you if needed.

## Install

```bash
curl -fsSL https://getvibekit.ai/install | sh
```

This downloads the binary for your platform and installs it to `~/.local/bin/vibekit`.

## Alpha Versions

To test upcoming features:

```bash
VIBEKIT_CHANNEL=alpha curl -fsSL https://getvibekit.ai/install | sh
```

Or install a specific version:

```bash
VIBEKIT_VERSION=cli-v0.1.6-alpha.1 curl -fsSL https://getvibekit.ai/install | sh
```

:::caution
Alpha versions may contain bugs. For production, use stable releases.
:::

## Manual Install

Download from [GitHub Releases](https://github.com/gabrielkuettel/vibekit/releases/latest), then:

```bash
chmod +x vibekit-<platform>
mv vibekit-<platform> ~/.local/bin/vibekit
```

## PATH Setup

If `~/.local/bin` isn't in your PATH:

<Tabs>
  <TabItem label="Bash">
    ```bash
    echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc && source ~/.bashrc
    ```
  </TabItem>
  <TabItem label="Zsh">
    ```bash
    echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc
    ```
  </TabItem>
  <TabItem label="Fish">
    ```bash
    fish_add_path ~/.local/bin
    ```
  </TabItem>
